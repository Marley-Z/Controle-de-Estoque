Projeto Gestão de Estoque:
Aplicação desktop desenvolvida em Java com o objetivo de gerenciar produtos em estoque, permitindo o controle de quantidades, organização em prateleiras e visualização de itens de forma simples. O projeto foi desenvolvido como parte de estudos em desenvolvimento de software, focando na integração com banco de dados e construção de interfaces gráficas.

Objetivo do Projeto:
Este projeto foi criado com fins educacionais e acadêmicos, com o objetivo de praticar:
- Desenvolvimento em Java
- Integração com banco de dados MySQL
- Modelo MVC
- Estruturação de aplicações desktop
- Manipulação de dados (CRUD)
- Uso de interface gráfica com Swing

Funcionalidades:
- Cadastro de produtos
- Controle de estoque
- Gerenciamento de prateleiras
- Associação de produtos às prateleiras
- Listagem de itens cadastrados

Tecnologias Utilizadas:
- Java
- MySQL
- XAMPP (phpMyAdmin)
- Java Swing

Instalação e Execução:
- Clone o repositório:
- Inicie o XAMPP (MySQL)
- Acesse o phpMyAdmin e importe o arquivo: database\controle_estoque.sql
- Execute a classe Inicio.Java
